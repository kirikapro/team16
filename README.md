DOWNLOADING AND RUNNING INSTRUCTIONS

1.  Make sure you are using an up-to-date mac 
2.  Install Xcode in the App Store if you haven’t installed it 
3.  Access our public repository using the link: https://github.com/kirikapro/team16.git  
4.  Press the green button "<> Code” button and then “Download ZIP” as shown below 
 
5.  Open the downloaded file “team16-main” and double-click on FeedTheFridge3.xcodeproj and then click on “Trust and Open” 
6.  Once opened, click the Play button at the top right of the screen 
7.  If it’s the first time downloading Xcode, you will need to download the simulator which you will be prompted to you after pressing the Play button. This will take you some time to download it.
